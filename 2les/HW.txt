Не строгое равентсво и сравнение работает по рапзному. При сравнении null преобразуется  в число 0. Поэтому выражение null >= 0 истинно, а null > 0 ложно.
Для нестрогого равенства == значений 0 и null действует особое правило: эти значения ни к чему не приводятся. Поэтому null == 0 ложно.
